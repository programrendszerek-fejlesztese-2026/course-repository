Íme néhány minta cURL hívás az ExpressJS végpontok teszteléséhez: --- **Kategóriák** - Kategóriák listázása: ``` curl -X GET http://localhost:3000/api/categories ``` - Új kategória létrehozása: ``` curl -X POST http://localhost:3000/api/categories \ -H "Content-Type: application/json" \ -d '{"name":"Saláta","description":"Friss zöldségek"}' ``` --- **Receptek** - Receptek listázása: ``` curl -X GET http://localhost:3000/api/recipes ``` - Receptek szűrése kategóriára: ``` curl -X GET "http://localhost:3000/api/recipes?category=1" ``` - Új recept létrehozása: ``` curl -X POST http://localhost:3000/api/recipes \ -H "Content-Type: application/json" \ -d '{"title":"Töltött paprika","description":"Hagyományos magyar étel","categoryId":"2","ingredients":[{"name":"paprika","quantity":"4 db"},{"name":"rizs","quantity":"100 g"}]}' ``` --- **Felhasználók** - Felhasználók listázása: ``` curl -X GET http://localhost:3000/api/users ``` - Új felhasználó létrehozása: ``` curl -X POST http://localhost:3000/api/users \ -H "Content-Type: application/json" \ -d '{"username":"geza","email":"geza@recept.hu","password":"geza123"}' ``` --- **Értékelések** - Értékelések listázása: ``` curl -X GET http://localhost:3000/api/ratings ``` - Új értékelés létrehozása: ``` curl -X POST http://localhost:3000/api/ratings \ -H "Content-Type: application/json" \ -d '{"recipeId":"1","userId":"2","score":5,"comment":"Nagyon finom!"}' ``` --- Ezekkel a cURL parancsokkal tesztelheted az összes fő CRUD végpontot.
